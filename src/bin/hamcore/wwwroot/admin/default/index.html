ComfyConnect VPN Server

Server Administration Console

Manage your Virtual Hubs, users, and connections. Set up secure work-from-home access for your team in minutes — all from your browser.

Signing in

This console authenticates with HTTP basic authentication:

  • Full server administrator: leave the username empty (or use administrator) and enter the server administrator password.
  • Virtual Hub administrator: enter the Virtual Hub name as the username and that hub's administrator password.

All actions on this page call the ComfyConnect VPN JSON-RPC API directly. You can also automate the server from your own apps (JavaScript, TypeScript, C#, Java, Python, Ruby, and more) — see the API reference.
